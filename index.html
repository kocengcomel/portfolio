01

OceanTrust: Marketplace for Marine Tourism

Final Year Project - SEATRU UMT (Sea Turtle Research Unit)

A comprehensive web-based snorkeling trip marketplace developed for SEATRU UMT (Sea Turtle Research Unit) as my Final Year Project. Built using PHP with CodeIgniter 4 framework and MySQL database, this platform connects customers with boat operators for marine tourism experiences. The system features dedicated boat operator dashboards where service providers can manage their trips, schedules, and bookings while supporting SEATRU's marine conservation and sea turtle protection initiatives.

Key Features

  • Customer registration and authentication system
  • Snorkeling trip browsing with advanced filters and search
  • Real-time booking and availability management
  • Boat operator dashboard for managing trips and schedules
  • Admin panel for overall platform management
  • Marine conservation educational content integration

Technology Stack

CodeIgniter 4 MySQL phpMyAdmin HTML5 CSS JavaScript Bootstrap MVC Architecture
